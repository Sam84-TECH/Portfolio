1er Projet

Objectifs

  • Sensibiliser à l’impact des comportements alimentaires et de l’activité physique sur la santé.
  • Proposer une solution technologique qui pourrait être utilisée à des fins de prévention personnalisée.

Outils utilisés

  • Python (langage principal)
  • Pandas / NumPy (manipulation de données)
  • Matplotlib / Seaborn (visualisation)
  • Scikit-learn (modèles de machine learning)
  • Jupyter Notebook (développement et documentation du code)
  • Django (création d’une interface web)
  • GitHub pour le travail collaboratif

Mars 2025

Resumé du projet

Durant la coding week organisé par l'ecole centrale-casablanca, nous avons a 5 travaillé sur un projet de machine learning visant à prédire le niveau d'obésité d'une personne en fonction de données comme l'âge, le sexe, les habitudes alimentaires, l'activité physique, la consommation d’alcool, de fast-food, ou encore les heures de sommeil.

L’équipe a commencé par analyser un jeu de données réel, effectuer un nettoyage des données, une analyse exploratoire, puis entraîner plusieurs modèles de classification (notamment des arbres de décision, des forêts aléatoires et des modèles de régression logistique). Le modèle le plus performant a été sélectionné en fonction de sa précision et de sa capacité à généraliser sur des données de test.

Enfin, une interface utilisateur simple a été conçue pour permettre à une personne d’entrer ses informations et de recevoir une prédiction personnalisée.
